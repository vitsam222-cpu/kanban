# Финансовая Kanban-доска

Минималистичное одностраничное приложение без сервера для ведения клиентов в формате финансовой kanban-доски.

## Что умеет MVP

- создаёт шесть колонок по умолчанию;
- позволяет добавлять, переименовывать, удалять и перетаскивать колонки;
- позволяет создавать, редактировать, удалять и перетаскивать карточки клиентов;
- автоматически считает суммы по колонкам;
- хранит данные в `localStorage`;
- поддерживает поиск по клиентам, услугам и контактам;
- использует тёмную адаптивную тему.

## Запуск локально

```bash
python3 -m http.server 4173
```

После запуска откройте `http://localhost:4173`.

## Автопроверка и деплой

- workflow `.github/workflows/verify.yml` проверяет синтаксис `src/main.js` и следит, чтобы в хедере не появлялись дубли кнопки, поиска, фильтра и старого блока баланса;
- workflow `.github/workflows/deploy-pages.yml` автоматически публикует текущую статическую версию сайта на GitHub Pages после push в `main`.

## Деплой

Для GitHub Pages включите в настройках репозитория режим **GitHub Actions** в разделе **Pages**. После этого каждый push в `main` будет публиковать актуальную версию приложения автоматически.
