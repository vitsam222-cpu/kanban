# Финансовая Kanban-доска

Минималистичное одностраничное приложение без сервера для ведения клиентов в формате финансовой kanban-доски.

## Что умеет

- 6 стартовых колонок;
- добавление, переименование, удаление и перетаскивание колонок;
- создание, редактирование, удаление и перетаскивание карточек;
- поиск по клиентам, услугам и контактам;
- автоматический подсчёт сумм по каждой колонке;
- сохранение данных в `localStorage`.

## Запуск локально

```bash
python3 -m http.server 4173
```

Откройте `http://localhost:4173`.

## Деплой

Проект является статическим сайтом. Его можно загрузить на GitHub Pages, Netlify или Vercel без дополнительной сборки.
